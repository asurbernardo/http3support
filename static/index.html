What is HTTP/3?

It is the new Hypertext Transfer Protocol standard developed by Google and backed by Cloudflare and Mozilla.

H3 uses QUIC instead of TCP, which provides some benefits over its predecesor like reduced connection latency (0-RTT for known servers), connection migrations and better congestion control.

What is this site?

Basically I packaged Cloudflares Quiche HTTP3 client, built a web interface around it and deployed it to the internet. You can check if your server supports H3 protocol. This is an Open Source project so you can audit the code and contribute if you feel like it. You can check the Github repository here.

How can I support HTTP/3?

If you ask me, by far the easiest way is to use Cloudflare as a proxy to your site (this is what I do), if you feel a bit more crafty there are experimental implementations for Nginx. Have fun!
